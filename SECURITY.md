# Security Policy

## 対象範囲

- このリポジトリのコードと設定
- ローカル実行前提のベータ版構成

## 報告方法

脆弱性が疑われる内容は、公開 Issue ではなくメンテナーへ非公開で連絡してください。  
再現手順、影響範囲、暫定回避策があれば併記してください。

## 既知の前提

- 本プロジェクトは単一ユーザーのローカル実行を前提としています。
- 認証・認可は未実装です。
- 本番公開環境での利用はサポート対象外です。

## 対応方針

- 受領後、再現確認とトリアージを実施
- 影響が高いものから優先して修正
- 必要に応じて README / 変更履歴で告知
