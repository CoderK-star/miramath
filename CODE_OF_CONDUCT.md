# Code of Conduct

## Our Pledge

私たちは、年齢、体型、障がい、民族、性自認、経験レベル、国籍、外見、宗教、性的指向に関わらず、
誰にとっても参加しやすいコミュニティを目指します。

## Our Standards

歓迎される行動:

- 建設的で敬意のあるコミュニケーション
- フィードバックを受け入れる姿勢
- 学習者と貢献者の多様な背景を尊重する態度

許容されない行動:

- 嫌がらせ、侮辱、差別的発言
- 個人攻撃や継続的な挑発
- 公開・非公開を問わない不適切な情報の晒し

## Enforcement

問題行動を見つけた場合はメンテナーに報告してください。  
重大な場合、警告・一時停止・参加拒否などの対応を行います。
